## Design Guidelines

### Typography
- **Headline Font:** Bold, sans-serif (e.g., Inter Bold)
- **Body Font:** Regular, sans-serif (e.g., Inter Regular)
- **Font Size:** 10-12pt body, 14-16pt headlines

### Colors
- **Primary:** RoMarketCap brand color
- **Text:** Dark gray (#1a1a1a)
- **Accents:** Light gray (#f5f5f5) for backgrounds

### Visuals
- **Screenshots:** High-quality, annotated if needed
- **Icons:** Consistent style, simple
- **Charts:** If including data, use simple bar/line charts

### Layout
- **Grid:** 2-3 column grid
- **Whitespace:** Generous margins and spacing
- **Alignment:** Left-aligned text, centered headlines
