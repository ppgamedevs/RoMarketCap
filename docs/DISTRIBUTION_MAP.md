# Distribution Map (Non-Paid Channels)

**Last Updated:** 2024  
**Status:** Strategy Locked

---

## Overview

RoMarketCap's distribution strategy focuses on organic, non-paid channels. We own attention through valuable content, community engagement, and strategic platform presence. This document defines platform-by-platform strategies, posting rules, and anti-spam guidelines.

---

## Core Distribution Principles

1. **Value First** - Always provide value, never spam
2. **Community Engagement** - Participate authentically, don't just promote
3. **Data-Driven** - Share insights, not opinions
4. **Transparent** - Disclose affiliation, be honest
5. **Respectful** - Follow platform rules and community norms

---

## Platform Strategies

### Reddit

#### Target Subreddits
- `/r/Romania` - General Romanian community
- `/r/rostartups` - Romanian startup community
- `/r/investing` - Investment discussions (if relevant)
- `/r/entrepreneur` - Entrepreneurship (if relevant)

#### Posting Strategy
- **Frequency:** 1-2 posts per week maximum
- **Content Types:**
  - Data insights ("Romania has X companies in software")
  - Market analysis ("Top 10 software companies by ROMC score")
  - Educational content ("How ROMC scores work")
- **Format:** Text posts with data, not link dumps

#### Posting Rules
- ✅ **DO:** Provide value, cite sources, engage in comments
- ✅ **DO:** Answer questions helpfully
- ✅ **DO:** Share insights, not just links
- ❌ **DON'T:** Post direct links to RoMarketCap (use text posts)
- ❌ **DON'T:** Post more than 2x per week per subreddit
- ❌ **DON'T:** Ignore negative feedback
- ❌ **DON'T:** Use multiple accounts

#### Anti-Spam Guidelines
- **90/10 Rule:** 90% value, 10% promotion
- **No link dumping:** Always provide context
- **Engage authentically:** Respond to comments, participate in discussions
- **Respect mods:** Follow subreddit rules, don't argue with moderators

#### What NOT to Post
- Direct links to homepage (use text posts with data)
- Promotional content without value
- Repeated posts across multiple subreddits
- Low-effort content ("Check out RoMarketCap!")

#### How to Respond
- **To questions:** Answer helpfully, provide sources
- **To criticism:** Acknowledge, explain, offer to improve
- **To spam accusations:** Be transparent, show value provided
- **To mods:** Respect decisions, don't argue

#### When to Disengage
- If post is removed by mods (don't repost)
- If receiving consistent negative feedback
- If community clearly doesn't want this content
- If accused of spam (apologize, step back)

## Cross-Platform Best Practices

### Content Adaptation
- **Reddit:** Text-focused, data-heavy, community-oriented
- **LinkedIn:** Professional, visual, network-oriented
- **Twitter:** Concise, visual, conversation-oriented
- **HN:** Technical, value-focused, discussion-oriented
- **Facebook:** Community-focused, visual, discussion-oriented

### Timing
- **Reddit:** Weekday mornings (9-11 AM Romania time)
- **LinkedIn:** Weekday mornings (8-10 AM Romania time)
- **Twitter:** Weekday mornings and afternoons (9 AM, 2 PM Romania time)
- **HN:** Weekday mornings (9-11 AM US Pacific time)
- **Facebook:** Weekday evenings (6-8 PM Romania time)

### Engagement
- **Response time:** Within 24 hours
- **Tone:** Helpful, authentic, professional
- **Frequency:** Don't over-engage (appears spammy)
- **Value:** Always provide value in responses

## Anti-Spam Guidelines (Universal)

### The 90/10 Rule
- **90% value:** Educational content, insights, data
- **10% promotion:** Soft mentions, CTAs, links

### Content Quality
- **Always provide value:** Never post without insights
- **Cite sources:** Always cite data sources
- **Be transparent:** Disclose affiliation clearly
- **Respect platforms:** Follow platform rules

### Engagement Rules
- **Respond authentically:** Don't use templates
- **Provide value:** Answer questions helpfully
- **Respect feedback:** Acknowledge criticism
- **Don't argue:** Accept mod/admin decisions

### Frequency Limits
- **Reddit:** Max 2x per week per subreddit
- **LinkedIn:** Max 1x per day
- **Twitter:** Max 2x per day
- **HN:** Max 2x per month
- **Facebook:** Max 2x per month per group

## How to Respond (Universal)

### To Questions
- **Answer helpfully:** Provide value, cite sources
- **Be authentic:** Don't use templates

- **Follow up:** Check if answer was helpful

### To Criticism
- **Acknowledge:** Don't ignore or delete
- **Explain:** Provide context and reasoning
- **Improve:** Use feedback to improve
- **Apologize:** If mistake was made

### To Spam Accusations
- **Be transparent:** Show value provided
- **Explain:** Provide context
- **Respect:** Accept if community doesn't want content
- **Disengage:** Step back if needed

### To Mods/Admins
- **Respect decisions:** Don't argue
- **Follow rules:** Adhere to platform rules
- **Ask permission:** If unsure about posting
- **Thank:** Acknowledge their work

## Crisis Management

### If Accused of Spam
1. **Acknowledge:** Don't ignore or delete
2. **Explain:** Provide context and value
3. **Apologize:** If mistake was made
4. **Disengage:** Step back if needed
5. **Learn:** Use feedback to improve

### If Post is Removed
1. **Don't repost:** Accept mod/admin decision
2. **Review:** Check if content violated rules
3. **Improve:** Adjust content for future
4. **Respect:** Don't argue with mods/admins

### If Receiving Negative Feedback
1. **Listen:** Acknowledge feedback
2. **Respond:** Address concerns authentically
3. **Improve:** Use feedback to improve
4. **Disengage:** If feedback is consistently negative
